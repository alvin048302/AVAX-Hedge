How to Get Your API Key
1️⃣
Go to console.anthropic.com
This is separate from claude.ai. Create a new account or sign in with your email.
2️⃣
Add a credit card
Go to Billing → Add payment method. You get $5 free credit. At ~$0.007 per analysis that covers 700+ analyses — years of nightly use.
3️⃣
Create an API key
Click "API Keys" → "+ Create Key" → name it "AVAX Advisor" → copy it immediately (shown only once).
4️⃣
Paste it above
Paste the key in the field above and tap Save Key. Done — tap Analyse to use it.
How the Analysis Works
🔍
Live web search every time
When you tap Analyse, Claude searches the web for current AVAX price, RSI, MACD, support/resistance levels, BTC trend, and Fear & Greed. Fresh data every night.
📐
S/R anchored LP range
LP range low = nearest real support level. LP range high = nearest real resistance. Range updates every analysis based on current market structure.
⚡
Trade signal with confidence
Long/Short/Wait based on LP zone position + RSI + MACD + BTC trend. Confidence % tells you whether to trade full size, half size, or skip tonight.
🔒
Confidence levels
70%+ HIGH → trade $1,000 at 2×. 50–70% MEDIUM → trade $500 at 2×. Below 50% WAIT → no trade tonight.
